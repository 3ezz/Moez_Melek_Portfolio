Roadkill

UE5 2.5D Blueprint In Progress

Roadkill is my end-of-year project currently in active development. I’m building all gameplay systems in Blueprint, refining level design continuously, and crafting the complete UI/UX flow from menus to in-game cinematics.

Gameplay Demo

Demo clip: in-game cinematic sequence with typewriter timing and in game abilities.

Overview

This is a 2.5D Unreal Engine 5 project focused on strong game feel, readability, and cinematic pacing. The core goal is to combine responsive mechanics with clean visual communication and polished progression flow.

Key Features

  • Gameplay mechanics fully implemented in Blueprints
  • Custom camera travelling system adapted to a 2.5D gameplay space
  • Typewriter effect sequences for intro and in-game cinematic moments
  • Ongoing level design iteration to improve pacing and player guidance

My Role

  • Designed and implemented all core mechanics in Blueprint
  • Built the full UI/UX pipeline (main menu, settings, in-game interfaces)
  • Created cinematic text presentation with typewriter effects
  • Continuously updated level design based on playtest feedback
