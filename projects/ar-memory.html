AR Memory

Unity AR Mobile UI/UX

Postcards come to life in AR. A lightweight mobile prototype focused on clean UI flow, AR triggers, and 3D presentation.

Overview

AR Memory is an AR prototype where printed cards trigger 3D content. It’s designed to feel simple, magical, and quick to understand.

Key Features

  • Card/marker detection → AR content reveal
  • Gallery-style flow to browse experiences
  • UI focus: readability + minimal steps to launch AR

My Role

  • UI design + integration
  • AR module integration
  • 3D asset integration + scene setup
  • Gallery module + navigation flow
